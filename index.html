About

From layout to logic.

I'm a front-end developer who likes projects small enough to ship in a weekend and useful enough to keep open in a tab. Calculators, timers, EMI tools, login flows — the everyday utilities that reward good UX far more than they reward flash.

Lately I've been pushing past the browser: learning Node.js, Express, and databases so the tools I build can hold state, talk to a server, and survive a page refresh. On the visual side, Figma, Photoshop, and Illustrator let me design the interface before I write a line of CSS for it.
